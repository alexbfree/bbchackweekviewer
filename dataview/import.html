1. Import your data

Import your data

In order to use your personal data store you'll need to fill it with your data.
On this page we'll guide you through how to import the files you have downloaded from Facebook, Google and elsewhere so that we can help you get your hands on your juicy data!

 It's safe. The data that you import will ONLY be stored in your web browser, never uploaded to the web or seen by the BBC.
You can delete it again whenever you like by clearing your browser or using our delete buttons.

 Facebook  Ready for data.  {{facebookMessageInboxList.length}} message threads found in Inbox.

The data we can currently load from Facebook is your messages directory which contains many JSON files. If you haven't already, you can download this from Facebook's Download Your Information tool. You can click the "How to get your data" link at the bottom of this page for detailed instructions. Make sure you download in JSON not HTML format.

Where is your Facebook data?

Please unzip your Facebook JSON download and browse to the directory containing your unzipped data. The folder to click 'Upload' on is the one which contains the messages folder, and it's probably called something like facebook-yourname.
